Java使用栈写简单逆波兰计算器思想及解释

数据结构基础
发布日期:   2022-04-25
文章字数:   1.1k

逆波兰计算器实现

逆波兰计算器实现表达式计算

     public static int cacul(List<String> list){
            //定义一个数值栈
            Stack<String> num = new Stack<>();
            //遍历得到的list,如果是数据就push到栈中，如果是符号，取出栈顶的元素进行基本操作
            for (String e : list){
                //正则表达式
                if(e.matches("\\d+")){
                    num.push(e);
                }else{
                    int num1 = Integer.parseInt(num.pop());
                    int num2 = Integer.parseInt(num.pop());
                    int res = 0;
                    if (e.equals("+")) {
                        res = num1 + num2;
                    }else if (e.equals("-")){
                        res = num2 - num1;
                    }else if (e.equals("*")){
                        res = num1 * num2;
                    }else if (e.equals("/")){
                        res = num2 / num1;
                    }else{
                         return 0;
                    }
                    num.push(res + "");
                }
    
            }
            //返回计算结果
            return Integer.parseInt(num.pop());
        }

中缀表达式变为后缀表达式（逆波兰）实现

实现思想

  • 数字压入数字栈（遍历pop出（即反向存储为正序））

  • 符号，当遇到“（”时开始到“）”结束，中间所有符号暂时压入符号栈，遇到“）”后，将符号pop出栈到数字栈，直到删除至“（”；
    其他符号看优先级，当前符号的优先级大于栈顶的符号优先级，则当前符号入符号栈，否则符号栈的符号出栈保存到数字栈，直到比当前符号的优先级高；
    符号栈剩余的符号全部pop出至数字栈

  • 举个栗子 例如A+B*(C+D)-E
    1.A为数字压入数字栈 数字栈：A
    2.“+”号压入符号栈 符号栈：+
    3.B为数字压入数字栈 数字栈：A，B
    4.“”为符号，且优先级大于+ 压入符号栈 符号栈：+，
    5.“（”左侧括号压入符号栈 符号栈：+，，（
    6.C为数字压入数字栈 数字栈：A，B，C
    7.“+” 入符号栈 符号栈：+，    ，（，+
    8.D为数字压入数字栈 数字栈：A，B，C，D
    9.“）”为右括号，将“（”及中间符号出符号栈，压入数字栈，此时 数字栈：A，B，C，D ，+
    符号栈：+，，
    10.“+”为符号栈，“-”的优先级比符号栈的“    ” 低，所以“”出栈到数字栈，数字栈：A，B，C，D ，+，
    符号栈：+，
    “-”的优先级比符号栈的+相等，所以“+”出栈到数字栈，数字栈：A，B，C，D ，+，*
    符号栈：
    “-”号压入符号栈 符号栈：-
    11.E为数字，压入数字栈 数字栈：A，B，C，D ，+，，E
    12.没有数据了，将符号栈的数据压入数字栈 A，B，C，D ，+，    ，E，-

实现代码

  • 中缀表达式转换成为中缀数组，便于遍历

      public static List<String> midList(String str) {
          if (str == null) {
              return null;
          }
          //去掉开头结尾的空格
          str = str.trim();
          List<String> list = new ArrayList<>();
          char c = ' ';
          String s = "";
          for (int i = 0; i < str.length(); i++) {
              c = str.charAt(i);
              //为符号
              if (c < 48 || c > 57) {
                  //1.当遇见符号时，说明数字结束，数字加入集合，并将数字串置空
                  //最后一个符号后面的数字将在2.中添加
                  list.add(s);
                  s = "";
                  list.add(String.valueOf(c));
                  continue;
              } else {
                  s += c;
              }
          }
          //2.将最后一个数值放入
          list.add(s);
          return list;
      }

  • 中缀表达式转换成为逆波兰表达式

      public static List<String> midChangeLast(List<String> list) {
          if (list.size() == 0) {
              return null;
          }
          List<String> newList = new ArrayList<>();
          String str = "";
          //数据栈
  //        Stack<String> num = new Stack<>();
          //符号栈
          Stack<String> oper = new Stack<>();
          for (int i = 0; i < list.size(); i++) {
              str = list.get(i);
              if (str.matches("\\d+")) {
                  newList.add(str);
              } else if (str.equals("(")) {
                  oper.push(str);
              } else if (str.equals(")")) {
                  while (!oper.peek().equals("(")) {
                      newList.add(oper.pop());
                  }
                  oper.pop();
              } else {//比较运算符的优先级，如果该运算符的优先级大于栈顶的运算符，该运算符入栈，否则栈顶运算符出栈
                  while (oper.size() != 0 && Operation.comp(str,oper.peek()) <= 0){
                      newList.add(oper.pop());
                  }
                  oper.push(str);
              }
          }
          while(oper.size() != 0){
              newList.add(oper.pop());
          }
  
          return newList;
      }

  • 符号优先级比较类Operation

      class Operation {
      public static int comp(String str1, String str2) {
          return change(str1) - change(str2);
      }
      public static int change(String str) {
          int res = 0;
          switch (str) {
              case "+":
              case "-":
                  res = 1;
                  break;
              case "*":
              case "/":
                  res = 2;
                  break;
              default:
                  break;
          }
          return res;
      }
  }

结果展示

      String str = "24+5*6-70/2+7*6+7+5-6*2-10";
            System.out.println(cacul(midChangeLast(midList(str))));
            
            
      结果：51
